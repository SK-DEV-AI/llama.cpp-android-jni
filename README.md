# Llama.cpp Android JNI Bindings

Production-ready JNI bindings for running LLMs on Android using llama.cpp.

## Features

- ✅ 97/106 llama.cpp APIs implemented (92%)
- ✅ Complete text generation with 20+ sampling parameters
- ✅ LoRA adapter support
- ✅ KV cache management
- ✅ State save/load
- ✅ Embeddings extraction
- ✅ Grammar constraints (GBNF)
- ✅ Fill-in-Middle (FIM)
- ✅ Thread configuration
- ✅ Performance metrics

## Structure

- `jni-bindings/` - JNI C++ code and Kotlin wrappers
- `llama.cpp/` - Integrated llama.cpp source
- `docs/` - Documentation
- `example-app/` - Minimal Android example (coming soon)
